# Existing Core Orientation Tools - Research

## Overview

This document catalogs existing tools and solutions for measuring core alpha and beta angles in geological field work.

## Commercial Tools

### 1. Kenometer Core Orientation Tool
**Manufacturer**: Deakin Industries  
**Website**: https://deakin.com/products/kenometer-core-orientation-tool-hq-size

**Features**:
- Constructed from anodized aluminum for durability
- Laser-etched markings for permanent, precise measurements
- Available in NQ2 and HQ sizes
- Conversion kits available for other core sizes
- Allows simultaneous measurement of alpha and beta angles
- Lightweight and portable

**Key Insights**:
- Physical tool with manual measurement
- Requires visual reading of angles
- Durable construction suitable for field use

### 2. Discoverer® Goniometer (Ezy-Logger)
**Manufacturer**: Dynamics G-Ex  
**Website**: https://www.dynamicsgex.com.au/products/goniometer-core-orientation-tool

**Features**:
- Quick-reference tool for alpha and beta angle measurement
- Made from durable plastic
- Protractors in blue and red for easy identification
- Compact and lightweight design
- Provides alpha angle measurements to within 5 degrees
- Available in multiple core sizes: BQ, NQ, HQ, PQ

**Key Insights**:
- Color-coded protractors for visibility
- Accuracy: ±5 degrees for alpha angle
- Simple, manual measurement tool

### 3. StructMaster™
**Manufacturer**: SPT  
**Website**: https://sptab.com/structmaster/

**Features**:
- Digital core logger with image-based logging
- Automates measurement calculation
- High-resolution image capture
- Calculates alpha and beta angles using precision software
- Cloud platform integration (SmartCloud™)
- Allows recalculation of angles from captured images
- Reduces user dependency and human error
- Full traceability

**Key Insights**:
- **Digital solution** with automated calculations
- Image-based verification and audit trail
- Cloud integration for data management
- Portable and user-friendly

### 4. CoreMaster™
**Manufacturer**: SPT  
**Website**: https://sptab.com/coremaster/

**Features**:
- Digital core orientation system
- Adaptable to multiple core diameters with interchangeable adapters
- Bluetooth connectivity for instant data transmission
- SmartCloud™ integration for data management
- Field-replaceable batteries
- Real-time sensor checks for quality assurance

**Key Insights**:
- **Wireless data transmission** (Bluetooth)
- Digital sensors for measurement
- Cloud-based data management
- Battery-powered, portable device

## Comparison Matrix

| Tool | Type | Accuracy | Data Transfer | Portability | Cost Range |
|------|------|----------|---------------|-------------|------------|
| Kenometer | Manual | Visual reading | Manual entry | High | Low-Medium |
| Discoverer Goniometer | Manual | ±5° (alpha) | Manual entry | High | Low |

| CoreMaster | Digital | Sensor-based | Bluetooth | High | High |

## Key Takeaways for GeoPen Design

1. **Digital Advantage**: Digital tools (StructMaster, CoreMaster) offer automated measurement and data transfer
2. **Portability**: All tools prioritize lightweight, portable design for field use
3. **Data Integration**: Modern tools integrate with cloud platforms for data management
4. **Accuracy**: Digital solutions reduce human error through automation
5. **Form Factor**: Manual tools are simple and compact; digital tools are larger but more capable
6. **Market Gap**: No existing "pencil-like" form factor device with digital measurement and data transfer

## Opportunities for GeoPen

- **Unique Form Factor**: Pencil-like design for intuitive, portable use
- **Digital Measurement**: Automated angle calculation with sensors
- **Instant Data Transfer**: Wireless connectivity to mobile/tablet applications
- **Cost-Effective**: Balance between manual tools (low cost) and high-end digital solutions
- **User-Friendly**: Simple interface for quick field measurements
